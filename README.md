A modern, full-stack inventory management solution built with Django and TailwindCSS for small to medium businesses.

---

## ✨ Features

- ✅ Add, update, and delete products
- ✅ Track stock levels in real-time
- ✅ Manage product categories
- ✅ Generate inventory reports
- ✅ User authentication and authorization
- ✅ Responsive UI using TailwindCSS
- ✅ Low stock alerts

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Django 4.x |
| Frontend | TailwindCSS 3.x |
| Database | SQLite / MySQL |
| Templates | Django Templates |

## ⚡ Installation

### Prerequisites

- Python 3.10+
- pipenv
- Node.js & npm

### Setup Steps

```bash
# 1. Clone the repository
git clone https://github.com/faridi-nayunda/Inventory-Management-System-V1.git
cd Inventory-Management-System-V1

# 2. Install Python dependencies
pipenv install
pipenv shell

# 3. Install TailwindCSS dependencies
npm install

# 4. Apply database migrations
python manage.py migrate

# 5. Create admin user
python manage.py createsuperuser

# 6. Run development server
python manage.py runserver

# 7. Open browser at http://127.0.0.1:8000
